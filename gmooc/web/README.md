### 캡쳐 작업 양식
- 배경(테두리)색 : window 배경 짙은남색
- 윈도우 디스플레이 배율 : 125%
- 해상도 : 1920X1080
- 캡쳐 툴 : 픽픽 에티터

- 실행 결과 캡쳐 : 적당한 여백을 두고 콘텐츠만 캡쳐 후 에디터를 사용하여 그림자 효과 추가
- 전체 화면 상태에서 프로그램 다룰 때 : 1280X800
